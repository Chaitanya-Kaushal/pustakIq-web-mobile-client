# @pustakiq/mobile

The PustakIQ React Native app (Android + iOS), part of the PustakIQ monorepo.

**Setup, run, fonts/icons and troubleshooting are documented in the repo root
[README](../../README.md).** Always run `npm install` from the repo root, not
here — the app depends on the `@pustakiq/shared` and `@pustakiq/theme`
workspace packages.

Quick reference (from repo root):

```bash
npm run mobile           # start Metro
npm run mobile:ios       # build & run iOS
npm run mobile:android   # build & run Android
```

Source lives in [`src/`](src/): `auth/`, `components/`, `navigation/`,
`screens/`, `utils/`.
